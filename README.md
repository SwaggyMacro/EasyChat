# EasyChat - 你的桌面翻译/交流神器

EasyChat 是一款基于 .NET 和 Avalonia UI 开发的现代化桌面翻译工具。

本项目重构自开发者 2018 年高中时期使用易语言开发的“交流神器”软件。历经 8 年时间的沉淀与 3 次重构，现在的 EasyChat 基于全新的技术栈，旨在提供更流畅、更智能的跨语言交流体验。

## ✨ 核心功能

软件核心专注于解决两大场景的跨语言障碍：

1.  **截图翻译**：
    *   调用本地 PaddleOCR 进行高精度文本识别。
    *   自动翻译选中区域的文字，并在屏幕上直接覆盖显示译文，阅读体验无缝衔接。
2.  **输入翻译**：
    *   在任意聊天窗口按下快捷键，呼出输入框。
    *   输入母语（如中文），软件自动翻译为目标语言（如英文）。
    *   **自动投递**：翻译完成后，自动将译文发送到原本的对话框中，无需复制粘贴。

## 🤖 翻译引擎与 API 支持

出于开源性质，EasyChat 不提供内置的免费额度，你需要自行申请 API Key。我们推荐以下高性价比的渠道：

### AI 大模型 (推荐)
*   **[硅基流动 (SiliconFlow)](https://siliconflow.cn/)**：提供多种大模型，部分小参数量模型免费且足够用于翻译，速度快，效果好。

### 机器翻译
*   **[百度翻译开放平台](https://fanyi-api.baidu.com/product/11)**：个人认证用户每月享 100 万免费字符，企业用户 200 万。
*   **[腾讯云机器翻译](https://cloud.tencent.com/document/product/551)**：每月享有 500 万免费字符额度。

## � 使用教程

1.  **下载与安装**：
    *   在 [Releases](../../releases) 页面下载最新的 `EasyChat` 压缩包。
    *   解压后双击 `EasyChat.exe` 运行。

2.  **配置翻译源**：
    *   进入 **设置 (Settings)** 页面。
    *   配置你申请的 **大模型 (AI)** 或 **机器翻译** API Key。

3.  **快捷键设置 (关键的一步)**：
    *   **添加语种切换热键**：
        *   在设置中添加一个新的热键方案。
        *   选择翻译引擎（你配置好的源）。
        *   设置 **源语言** 和 **目标语言**。
        *   *💡 智能逻辑*：假设你设置源语言为**英文**，目标语言为**中文**：
            *   **截图翻译**时：识别屏幕上的**英文** -> 翻译成 **中文** (方便你看懂)。
            *   **输入翻译**时：你在输入框打**中文** -> 翻译成 **英文** (方便你回复)。
    *   **设置基础热键**：
        *   分别设置一个全局的“截图翻译热键”和“输入翻译热键”。

4.  **开始使用**：
    *   遇到看不懂的英文？按下**截图翻译热键**，框选它。
    *   想回喷/回复老外？按下**输入翻译热键**，打中文，自动变英文发出去。

## 🎮 进阶玩法：提示词 (Prompt) 优化

软件支持自定义 AI 模型的 System Prompt（系统提示词），这对于特定领域的翻译非常有用。

**举例 (CS2 游戏场景)**：
你可以添加一个针对 CS2 的提示词，告诉 AI 不要直译游戏术语。
> "这是一个 CS2 游戏内的对话翻译，若遇到术语请保留或使用游戏玩家习惯的黑话。例如不要将 'dinked' (爆头但没死) 翻译成 '打头一枪'。"

这样可以避免机翻闹笑话，实现真正的“本地化”交流。
